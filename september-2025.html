Our Speakers

Jan Finis & Adrian Vogelsgesang, Salesforce Hyper

Exploring Database Technologies in Salesforce Data Cloud

Previously in the Munich Database Meetup, the Hyper team shared the story of its incredible transition from an academic project and spin-off to an enterprise-grade database engine powering both Tableau and Salesforce. In this talk, we dig deeper into some of the key technologies that drove our success in Salesforce Data Cloud, an open data platform supporting Data Lake formats, AI agents, and a variety of zero-copy data integrations. We will explore the technical aspects of the core database features and SQL extensions that we have been building over the years to support such a rich data ecosystem and share some battle stories from the developer’s perspective.

Martin Milenkoski, AWS Redshift

Data Lake Query Optimization in Redshift

Amazon Redshift Spectrum allows users to query data in S3 without loading it into Redshift. This talk discusses query optimization techniques specifically designed for Spectrum queries, addressing unique challenges such as the lack of table and column statistics in external data sources. We present statistic-agnostic optimization techniques that improve performance without relying on traditional statistics and explore how integrating available statistics can further enhance query execution. Finally, we cover advanced optimization strategies that leverage both approaches to maximize Spectrum query performance across diverse data lake scenarios
